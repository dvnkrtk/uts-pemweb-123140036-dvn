## Deskripsi Proyek  

**Baca Nusantara** adalah aplikasi perpustakaan digital yang dirancang untuk memudahkan pengguna dalam mencari dan membaca buku secara daring. Aplikasi ini memanfaatkan **Open Library API** yang dapat diakses gratis tanpa memerlukan API key. Fitur-fitur seperti pencarian buku, daftar bacaan pribadi (*reading list*), dan tampilan detail buku dibuat dengan antarmuka modern berbasis **React.js** dan **Tailwind CSS**, memastikan tampilan yang ringan, responsif, dan mudah digunakan.

## Fitur Utama  

- **Pencarian Buku:** Cari buku berdasarkan *judul* atau *nama penulis*.  
- **Tampilan Cover:** Menampilkan sampul buku dari *Open Library API*.  
- **Reading List:** Tambahkan buku ke daftar bacaan pribadi.  
- **Hapus Buku:** Hapus buku dari daftar bacaan dengan mudah.  
- **Penyimpanan Lokal:** Data daftar bacaan disimpan otomatis menggunakan *localStorage*.

## Teknologi yang Digunakan  

| Teknologi | Keterangan |
|------------|-------------|
| **React.js** | Library JavaScript untuk membangun antarmuka interaktif |
| **Tailwind CSS** | Framework CSS modern dan responsif |
| **Open Library API** | Sumber data buku gratis tanpa API key |
| **React Hooks (useState, useEffect)** | Untuk pengelolaan state dan efek |
| **Vite** | Build tool untuk performa tinggi pada proyek React |
